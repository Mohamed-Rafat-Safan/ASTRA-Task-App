# ASTRA Task App

- This task for Astra company.
  
- In the Home screen it displays all posts.

- In the post details screen dispaly post details  with ability to edit or delete.
  
## The skills are used in this application:
- language: Kotlin </br>
- Navigation component </br>
- Material Design </br>
- View Binding </br>
- Retrofit2 </br>
- Clean Architecture </br>
- Architectural pattern by (MVVM) </br> 
- Live Data </br>
- Dependency injection by (Dagger Hilt) </br>
- Coroutines </br>
- DiffUtil </br>
- Glide </br>
- Intuit Libray </br> </br>
